# Legacy Python Implementation

This directory contains the original Python implementation of agent-deploy (v0.1.0).

## Reason for Archival

The project has been re-implemented in TypeScript under `node/` with significantly more features:
- 345+ tests, 100% pass rate
- 12 MCP tools (up from 4)
- Full runtime engine with YAML pipeline execution
- Agent execution with dynamic overrides
- Market integration (upload/download)

The Python version is kept for historical reference only. See `node/` for the active implementation.
